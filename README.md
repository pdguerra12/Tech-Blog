# Tech-Blog

## Purpose

To provide the user with a CMS-style blog site so they can publish articles, blog posts, their thoughts and opinions.

---

## Technology Used

- Express.js
- express-handlebars
- API
- MySQL2
- Sequelize
- Javascript

For authentication

- dotenv
- crypt
- express-session
- connect session-sequelize
